## 🚀 How to Use

### Option 1: Run Directly (Easiest for testing)
```powershell
# Double-click run_app.bat
# OR run in terminal:
python budget_planner.py
```

### Option 2: Build as .exe (Best for distribution)
```powershell
# Install PyInstaller
pip install pyinstaller

# Build the executable
python build_exe.py

# Your .exe will be in the 'dist' folder!
# You can share it with anyone - no Python needed!
```
