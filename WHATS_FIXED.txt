💡 HOW TO USE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ADD RECURRING EXPENSE:
1. Click "➕ Add Expense"
2. Fill details
3. Select "🔄 Recurring"
4. Save - Done!

EDIT AN EXPENSE:
1. See expense on dashboard
2. Click "✏️ Edit"
3. Make changes
4. Save or Delete

MANAGE INCOMES:
1. Click "💵 Manage Incomes"
2. See all incomes
3. Click "✏️ Edit" on any
4. Update or delete

VIEW RECURRING:
1. Click "🔄 Recurring Expenses"
2. See all monthly auto-expenses
3. Remove if needed
